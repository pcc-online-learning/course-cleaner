# course-cleaner
A widget for managing stylesheets throughout course webpages in D2L.

## Development
Develop locally with a python server. The repo is set up to use livereload
### Step 1 Clone the Repo

```bash
git clone <this-repo>
```
### Step 2 Create a Python Virtual Environment
The gitignore is already set up to ignore virtual environment files in the root.

```bash
cd <this-repo> && python -m venv .
```

### Step 3 Install Python Requirements
Currently this is only livereload.

```bash
pip install -r requirements.txt
```
